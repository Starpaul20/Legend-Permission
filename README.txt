Created by: Starpaul20
Copyright: ©2014
License: GPL

Adds a usergroup permission that allow you to disable of the edited by legend whenever you edit (please note that you may only disable the legend in full edit mode, it cannot be disabled in quick edit).

Please note that when you use this plugin, please make sure the settings 'Show 'edited by' Messages' and 'Show 'edited by' Message for Administrators' are set to yes (to prevent any collisions).

The usergroup permissions are located under the Editing/Deleting Options section of the 'Forums and Posts' tab when editing a usergroup.

This plugin offers full language support.


## INSTALLATION ##

1. Upload all files above, keeping the file structure intact.
2. Go to Configuration > Plugins
3. Click "Install & Activate"
4. Enjoy!


## UPDATING ##

Any beta version should be reinstalled.
